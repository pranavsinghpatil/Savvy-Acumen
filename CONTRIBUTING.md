# Contributing to Savvy-Acumen

First off, thank you for considering contributing to Savvy-Acumen! It's people like you that make Savvy-Acumen such a great tool for quick learning.

## How Can I Contribute?

### Adding New Learning Paths

1. **Choose a Domain**: Identify a technical domain that's not covered or needs updating
2. **Follow the Structure**:
   - Create a new directory with the domain name
   - Include all required sections (Prerequisites, Objectives, etc.)
   - Add practical examples and exercises
   - Keep content focused and concise

### Content Guidelines

- Content should be completable within one week
- Focus on practical, project-oriented learning
- Include real-world examples
- Provide clear, concise explanations
- Add references to official documentation
- Include code samples where applicable

### Quality Standards

- Ensure all content is accurate and up-to-date
- Check for grammar and spelling
- Verify that code examples work
- Include comments and explanations in code
- Test all provided exercises

## Submission Process

1. **Fork** the repository
2. Create a new **branch** for your changes
3. Make your changes following our guidelines
4. Test your changes thoroughly
5. Submit a **Pull Request** with a clear description of your changes

### Pull Request Guidelines

- Use a clear, descriptive title
- Explain what the changes accomplish
- Reference any related issues
- Include screenshots for UI changes
- Update documentation if needed

## Style Guide

### Writing Style

- Use clear, simple language
- Break down complex concepts
- Use bullet points and lists for clarity
- Include diagrams where helpful
- Keep paragraphs short and focused

### Code Style

- Follow language-specific best practices
- Include comments for complex logic
- Use consistent formatting
- Write self-documenting code
- Include error handling

## Need Help?

- Check existing issues and pull requests
- Open a new issue for discussions
- Tag maintainers for urgent matters
- Join our community discussions

## Code of Conduct

- Be respectful and inclusive
- Welcome newcomers
- Give constructive feedback
- Help others learn and grow
- Maintain professionalism

Thank you for contributing to making learning more accessible and efficient!

## üí° **Pro Tips**
1. **Reverse-Engineer Designs**: Use browser's "Inspect" tool on target websites  
2. **Consistent Spacing**: Use multiples of 4px (Tailwind's default scale)  
3. **60-30-10 Rule**: Primary color (60%), secondary (30%), accent (10%)  
4. **Glassmorphism**: Apply `backdrop-blur-lg bg-white/10` in Tailwind  
5. **Particle Effects**: [Particles.js](https://vincentgarreau.com/particles.js/) for background dynamics
